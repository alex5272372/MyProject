Install and Configure VNC server on Ubuntu 18.04:
-------------------------------------------------
$ sudo apt update
$ sudo apt install xfce4 xfce4-goodies
$ sudo apt install tightvncserver
$ vncserver
$ vncserver -kill :1
$ mv ~/.vnc/xstartup ~/.vnc/xstartup.bak
$ nano ~/.vnc/xstartup

#!/bin/bash
xrdb $HOME/.Xresources
startlxde &

$ sudo chmod +x ~/.vnc/xstartup
$ vncserver
